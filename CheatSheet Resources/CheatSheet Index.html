HTML Table Reference

In this section you will be reminded of the tags/elements of creating a table in html.

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

HTML Tag Reference

In this section you will be reminded of HTML tags. Some are shown in the table, but for more please click here!

HTML Tags

Tag Name Description
<!DOCTYPE> Declaration of document It is an information to the browser about what document type to expect.
<html> & </html> HTML Tag The root of an HTML document. It is the container for all other HTML elements (except for the <!DOCTYPE> tag.
<head> & </head> Head Element This is a container for metadata (data about data), placed between html tag and body tag.
<body> & </body> Documents Body This element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists etc.
<h1> - <h6> Headings h1 being the largest size and h6 being the smallest.
<p> & </p> Paragraphs This defines a paragraph. Browsers automatically add a single blank line before and after each <p> element.
<form> & </form> Form This tag is used to create an HTML form for user input. This element can contain one or more of the following form elements: <input>, <textarea>, <button>, <select>, <option>, <optgroup>, <fieldset>, <label>, <output>.
<input> Input Field This tag specifies an input field where the user can enter data. The <input> element is the most important form element.
